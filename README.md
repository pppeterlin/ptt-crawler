# ptt-crawler


## 用途
在PTT上各版（如[MacShop](https://www.ptt.cc/bbs/MacShop/index.html)、[Stock](https://www.ptt.cc/bbs/Stock/index.html)）設定有興趣的關鍵字後，自動排程追蹤包含關鍵字的推文，並發送至Gmail提醒。


## 開發重點
#### Python
* 爬蟲
* Gmail API
* multiprocessing
* multithreading 

#### Docker
* 打包dockerfile

#### Linux
* Shell Script
* crontab自動化排程

#### Database
* MongoDB
